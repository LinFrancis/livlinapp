# 🌿 LivLin · Indagación Regenerativa v7.4

**Potencial para una vida regenerativa** · www.livlin.cl

## Cambios v7.4

### Potenciales del Sitio integrados en Visión
La sección "🌿 Potenciales del Sitio" fue eliminada del sidebar. Las 10 dimensiones
(barras, definiciones, interpretaciones ERP/HRP) están ahora dentro de
"🌱 Visión y Estado Regenerativo → Perspectiva Comparada".

### Datos climáticos completos (API)
La sección "🔬 Observación Ecológica" ahora muestra TODOS los datos de API:
- Coordenadas geográficas (lat, lon, elevación)
- Gráfico climático anual (Open-Meteo — promedio histórico 5 años)
- Métricas clave: mes más cálido, mes más frío, T° máxima y mínima registrada
- Precipitación anual total
- Fuente de datos explícita

### Sección de Metodología completa
Nueva sección "📐 Metodología" dentro de Visión y Estado Regenerativo con:
- **¿Cómo se calcula el ERP?** — Fórmula 80% MFP + 20% M2-6, tabla de puntuación,
  detalle de cada sub-indicador con variables y escalas
- **¿Cómo se calcula el HRP?** — Fórmula 100% MFP proyectado
- **¿Qué es la Brecha?** — Definición + interpretación personalizada
- **Escala de niveles** — Tabla con los 5 niveles (Sin inicio → Abundancia)
- **¿Qué es la Flor de la Permacultura?** — 7 pétalos con subtítulos
- **¿Qué son las 10 Dimensiones?** — Cada dimensión con definición y fuentes

### Sidebar simplificado (8 secciones)
Visión · Datos · Tao · Ecología · Sistemas · Fotos · Síntesis · Bibliografía
